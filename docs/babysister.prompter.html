babysister.prompter module¶

babysister.prompter.query_yes_no(question, default='yes')¶

Ask a yes/no question via input() and return their answer.

Parameters

  • question (str) – string that is presented to the user.

  • default (str, ["yes" | "no" | None]) – presumed answer if the user just hits <Enter>. None implies an answer is required of user.

Returns

answer.

Return type

bool
